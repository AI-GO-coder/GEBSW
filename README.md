# Generalized Energy-based Sliced Wasserstein Distance (GEBSW)

Official implementation for the paper submitted to *Scientific Reports*.

## Code Structure

| File | Description |
|:---|:---|
| `PCR-15pairs-quanbushu.py` | Point cloud reconstruction on ModelNet40 (15 pairs, 18 variants, 10 repeats) |
| `heatmap-dragon-ding.py` | Ablation study: necessity of nonlinear projection and energy weighting |
| `COCO-ding.py` | Color transfer on COCO dataset |
| `DTD-ding.py` | Color transfer on DTD dataset |
| `WikiArt-ding.py` | Color transfer on WikiArt dataset |

## Requirements

- Python 3.8+
- PyTorch, NumPy, SciPy, pandas, matplotlib, trimesh,etc.
